# CONTINUITY REPORTS

Questa cartella contiene report di continuita pronti per ChatGPT.

## A cosa serve
- mantenere memoria operativa tra chat diverse
- avere report riassuntivi di stato aggiornati
- supportare il riavvio rapido del lavoro senza perdita di contesto

## Contenuto tipico
- contesto generale progetto/modulo
- stato attuale e ultime modifiche
- decisioni gia prese e vincoli da non rompere
- prossimo passo consigliato

## Regole operative
- usare il template `_TEMPLATE_CONTINUITY_REPORT.md`
- usare naming: `YYYY-MM-DD_HHMM_continuity_<area-o-modulo>.md`
- niente codice e niente diff
- marcare punti incerti con `DA VERIFICARE` o `NON TROVATO`
